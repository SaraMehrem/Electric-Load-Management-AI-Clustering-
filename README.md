A research-based project that applies AI clustering techniques to analyze and segment electricity consumption patterns using historical load data. The study uses K-means clustering to group demand into low, medium, and high consumption categories based on factors such as temperature and time, helping improve load forecasting, grid stability, and energy distribution. The project combines qualitative insights from expert interviews with quantitative data analysis, and evaluates model performance using metrics like the silhouette score to ensure meaningful and reliable clustering results.
